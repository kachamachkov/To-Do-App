# To-Do-App

## Summary:

This is a simple To-Do tracking browser app, created with HTMl/CSS/JS.

### Main Features:

-   Add a task (if invalid, alert user)
-   Track tasks and mark the completed ones
-   Delete done tasks
-   Saving the data after refresh
